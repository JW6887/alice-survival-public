# 跨境AI选品Agent

> 对话式AI选品助手。卖家说人话，Agent干全活。

## 一句话
"帮我找Q4欧美热销、客单价$20-40、轻小件可空运的厨房新品" → AI自动搜索、分析、推荐、匹配供应商、算利润。

## 为什么做这个
- 全球650万+活跃卖家，中国卖家占45%
- 现有工具都是**数据查询器**（表单→筛选→列表），不是**决策助手**
- 选品→供应链→利润，目前需要5-8个工具跳转
- LLM足够强，但没人做端到端Agent

## 竞品缺口
| 竞品 | 能力 | 缺失 |
|------|------|------|
| Jungle Scout | 选品数据库 | AI浅、仅Amazon |
| Helium 10 | 功能最全 | AI辅助、被动回答 |
| 鸥鹭/卖家精灵 | 中文体验好 | 无Agent能力 |
| Perci | AI原生 | 不成熟、仅Amazon |

**所有竞品的AI都是"回答问题"，没人"自主干活"。**

## MVP (Phase 1)
- Amazon单平台
- 对话式交互（自然语言→选品建议）
- 基础供应链匹配（产品→1688供应商→成本估算）
- 利润初步计算

## 技术栈
- LLM: DeepSeek/Claude API
- 数据: Amazon Product API + 1688开放平台
- 前端: 对话式Web UI
- 部署: GitHub Pages + Vercel
